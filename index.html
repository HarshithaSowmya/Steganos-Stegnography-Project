What is Steganography?

Steganography is the art and science of concealing information within a non-secret medium. Unlike cryptography, which scrambles a message, steganography hides the very existence of the communication.

Etymology & History

Derived from the Greek words "steganos" (hidden) and "graphein" (to write). From ancient hidden tattoos to digital pixel manipulation, it has been a core security tactic for centuries.

How It Works (LSB Method)

1 Binary Conversion: The secret data is converted into a binary string (0s and 1s).
2 Pixel Modification: The "Least Significant Bit" of each pixel color channel is replaced with a secret bit.
3 Visual Integrity: Because the change is so minute, the color shift is invisible to the human eye.

How to Use SteganoS

Step 1: Encryption

  • Select Encrypt from the Tools menu.
  • Upload a "Carrier Image" (the cover).
  • Type a message or upload a secret image.
  • Crucial: Set a password to lock the data.
  • Download the processed "Stego-Image".

Step 2: Decryption

  • Select Decrypt from the Tools menu.
  • Upload the image containing hidden data.
  • Enter the password used during encryption.
  • Click "Extract" to reveal the secret contents.
  • Download the revealed image if applicable.
Note: This tool uses pixel-level manipulation. For best results, use lossless formats like PNG. JPEG compression may corrupt hidden data.
